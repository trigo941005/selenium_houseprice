# selenium_houseprice

# 專案說明

## 技術關鍵字

    selenium
    time
    openpyxl

## 專案介紹

    此專案旨在從特定的房地產網站上爬取房價相關資訊，並將數據整理後存儲進excel 這些數據包括成交年月、地址、型態、總價、單價、建坪、地坪、樓別、屋齡、成交紀錄 這可以幫助用戶了解該地區的房價資訊

## 功能

    將數據儲存到excel表
    從指定的房地產網站爬取房價數據

## 使用資料

    此專案使用的數據包含5168房價時價登陸網
## 環境要求
Python 3.9.11
selenium 4.23.1
openpyxl 3.1.5
## 資料來源

    https://price.houseprice.tw/(5168房價時價登陸網)
